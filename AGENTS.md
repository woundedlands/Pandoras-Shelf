1. Project is web based cross platform html launcher. Allows installing any html as an app and manage them in windows
2. Tauri is used as cross platform framework, most of the app regular web app. It uses system webview
3. Libraries used: react, mantine, zustand, react-jss, tabler/icons-react
4. Typescript is in the strict mode
5. Project is organized to use this structure: src is entry, src/modules contain folders of modules
